# Security Policy

## Personal Tool Notice

**lit-critic (short for Literary Critic) is a personal tool maintained for individual use.**

This project is shared publicly for reference and learning purposes, but:
- **No security support is provided**
- **No vulnerability patches guaranteed**
- **Use at your own risk**

## Recommendations

If you use this tool:
- Keep your API keys secure (use environment variables, never commit them)
- Review the code yourself before running it
- Fork the repository if you need to maintain your own version

## Reporting

This is a personal project without formal security support. If you discover a critical security issue fork the repository and fix it.

For your own security:
- Never share your API keys
- Don't commit sensitive data to your fork
- Review any pull requests carefully if you accept them in your fork
